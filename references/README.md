# References

Drop reference material here once, and Claude can read it in **any** future
session — no need to re-upload in chat.

What to put here:
- Official curriculum / specification grids (PDF)
- Sample / model question papers (PDF or images)
- Photos or scans of past papers
- Style notes, marking guides, anything you want used as a reference

How to use it:
- Just say: "take a reference from the references folder" (or name a file,
  e.g. "look at references/see-maths-grid.pdf").
- Claude reads PDFs and images directly from here.

Tip: give files clear names (e.g. `see-science-curriculum-2081.pdf`,
`madhesh-2082-maths-paper.jpg`) so they're easy to point to.
